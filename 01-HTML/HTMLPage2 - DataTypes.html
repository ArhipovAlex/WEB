Типы данных в языке С++

Переменная - это именованная область памяти, содержимое которой может изменяться в процессе выполнения программы. Прежде чем ее использовать, ее нужно объявить следующим образом:

        
			type name;

type (тип переенной), определяет три вещи:

  1. сколько памяти переменная будет занимать;
  2. какие значения она может принимать;
  3. какие операции над ней можно выполнять;

Все типы данных в языке С++ можно разделить на следующие категории:

  1. Логические типы. Включает в себя всего один тип данных – bool (boolean). Переменная типа bool занимает 1 Байт памяти и принимает одно из двух значений: true или false. true хранится в памяти как 1, а false как 0. На самом деле 0 – это false, а true – это все что не 0;
  2. Числовые типы. Включает в себя два подтипа данных:

    Целочисленные типы:
    • short. Переменная типа short занимает 2 Байта памяти, и принимает значения в диапазоне
      unsigned short: 0...65 535; 0000 0000  0000 0000 ... 1111 1111  1111 1111; 
                0...216-1
signed short: -32 768...32 767; 
              -215...+215-1
    • long – занимает 4 Байта памяти, и принимает значения в диапазоне
      unsigned long: 0 ... 4 294 967 295;
               0 ... 232-1;
signed long: -2 147 483 648 ... 2 147 483 647;
             -231 ... +231-1;
    • int (Integer – Целое число). Платформенно зависимый тип данных, его величина зависит от процессора (CPU) операционной системы (ОС) и среды разработки (IDE – Integrated Development Environment). В Visual Studio для Microsoft Windows тип данных int занимает 4 Байта, следовательно, его диапазоны принимаемых значений полностью совпадают с long.
    • long long. Занимает 8 Байт памяти, и принимает значения в диапазоне 
      unsigned long long int: 0 ... 264-1;
signed long long int: -263 ... +263-1;

    Вещественные типы:

    Вещественные типы предназначениы для хранения дробных чисел, (чисел с плавающей запятой).

    Вехественные типы есть только знаковые, они не могут быть unsigned. В языке С++ есть всего два вещественных типа: float и double.

    float – вещественный тип одинарной точности, занимает 4 Байта памяти.

    double – вещественный тип двойной точности, занимает 8 Байт памяти.

    float и double могут хранить ОЧЕНЬ БОЛЬШИЕ и ОЧЕНЬ МАЛЕНЬКИЕ числа, но эти числе могут быть не совсем точными.

    Разделителем целой и дробной части у float и double является точка, а не запятая.
